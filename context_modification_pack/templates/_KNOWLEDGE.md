# _KNOWLEDGE.md — Static Project Memory

> **Stable facts only. Append when a fact is established; do not write speculative or in-flight content here.**
> **Sectioned for scoped reads — fetch by `##` header, do not load wholesale.**

## How to use this file

- Each section has a stable header. Read a section by its `##` anchor.
- New stable fact? Add to the right section, or create a new section with a stable name.
- In-flight reasoning, hypotheses, or current goals → those go in `_STRATEGY_STATE.md`, not here.
- When `_STRATEGY_STATE.md` references a section, it must use the `##` header verbatim so the slice is clean.
